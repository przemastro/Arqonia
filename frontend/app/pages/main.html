Getting started info

In the 'Observations' page guest user is able to search for active observation and edit selected observation in UVB filters. YES/NO flag informs if the observation was taken in appropriate filter.
In order to have write/modify permissions guest user is obligated to register (if it has not been done yet) and login. When user registers an email notification is sent with provided login details and welcome message. Logged in user after logging is redirected to the 'Home' page.
In the 'Observations' page user is able to add new observation. Basic info like star name, start date and end date are mandatory. In order to correctly add observation at least one filter observation is also required. After submitting form new observation is added to the stack of observations to be processed. It does not appear in the table as long as it is not processed.
System accepts csv files as presented below:
2720.81478,-4.971
2720.81479,-4.971
2720.81480,-4.981
2720.81481,-4.961
2720.81482,-4.971
Obviously user still can view UVB photometry.
User can edit existing active observation. By default some fields are pre-populated and some are not. In order to process at least one filter photometry needs to be added to the observation. When submitted observation is added to the stack of observations and waiting for processing. When editing for the second time it will still have old values. No worries new ones have been already submitted.
Finally user is able to remove photometry. When clicked yed it does not dissapear from the table list.
When user is done with all of observation he can process them in the 'Admin Panel'. Usually it takes few seconds for the table list to get update.
Last but not least logged in and guest users can view CMD diagram in the 'Diagrams' page.
