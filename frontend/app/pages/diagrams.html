  • Color Magnitude Diagrams

Color Magnitude Diagrams (CMD) version of Hertzsprung-Russell (HR) diagrams are available in following filters combinations:

V vs. B-V. Generated only if photometry was done in both B and V filters.
B vs. U-B. Generated only if photometry was done in both U and B filters.
I vs. R-I. Generated only if photometry was done in both R and I filters.
I vs. V-I. Generated only if photometry was done in both V and I filters.

  • Light Curves

For each object with photometry in UVBRI filters diagrams Flux vs. JD are generated. At least one observation needs to be present in order to generate plot. Flux is the absolute magnitude. JD is the standard Julian Date.
