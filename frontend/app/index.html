FREQUENTLY ASKED QUESTIONS

What kind of file/data should I load to the system?

System allows two columns .csv file without column headers. First column is an absolute magnitude, the second one is a JulianDate.

I have already done photometry by myself and I have .csv file. Can I skip 'Photometry' step in order to process data?

Yes. 'Photometry' step is optional. If you already done photometry and have correct .csv file you can load it to the system and process data.

Why I can't see new observation I have just added?

Everytime user is adding, removing or updating observation it is not immediately added to the list of observation. This give a time for user to revoke changes. To see changes in personal list and diagrams user needs to [Process] data firstly.

Newly processed observation is visible only in personal diagrams not in global.

User who is sure about the quality of the observation and performed photometry can mark [New Observation] or existing one as 'Verified'. That flag causes appearance of the observation in the global diagrams. If [No] is selected or nothing is selected then observation appears only in personal diagrams.

I can't find object in the 'Search' area using popular name e.g. 'Proxima Centauri'.

This is because system accepts only names present in the listed catalogs. E.g. comet 'Schwassmann-Wachmann' will not be found because it's catalogs name is '73P/Schwassmann-Wachmann'.
