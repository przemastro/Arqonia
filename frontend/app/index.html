  • 1. Login or Sign up

Before any data processing can be started an existing user needs to Login to the service. New user has to Sign up firstly.
  • 2. Load data

User can load new data or modify/remove existing data. System allows .csv files (flux, time) without column headers. Please see FAQ section.

  • 3. Process data

Once user is done with data loading it is time to process data. Be careful all observations marked as "verified" will appear not only in users diagrams but also in global diagrams.
  • 4. Verify results

Processing of data usually takes few seconds. When this is done its time to verify your results in diagrams.
